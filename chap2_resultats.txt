### Statistiques globale

Pour l'analyse globale des résultat nous allons considérer les différentes classes suivante :
- Vrai positif : pour les polymères dont s2m retrouve exactement la structure annotée dans la base de données correspondante.
- Faux positif : pour les polymères dont s2m trouve une couverture à 100% mais dont les monomères ne sont pas exactement les mêmes que dans la BDD.
- Faux négatif : pour les polymères dont s2m ne retrouve pas tous les monomères.

Nous soulignons le fait qu'il n'existe pas de faux négatif car nous n'avons pas d'annotations incomplètes dans les bases de données.
